# CSC580 - Distributed Data Analytics Using Message Passing Interface (MPI)

## Project Overview

This project was developed for the CSC580 course to compare the performance of sequential and distributed data analytics using Microsoft Message Passing Interface (MS-MPI). The system performs statistical analysis on datasets of different sizes and evaluates the effectiveness of parallel processing.

## Objectives

- Develop a sequential data analytics program.
- Develop an MPI-based distributed data analytics program.
- Compare the performance between sequential and MPI implementations.
- Evaluate execution time, speedup, efficiency, and parallel overhead.

## Technologies Used

- Microsoft MPI (MS-MPI)
- Visual Studio Code
